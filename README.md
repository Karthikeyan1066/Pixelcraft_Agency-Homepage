# Design Agency Homepage – Next.js Internship Task

## Overview

This project is a **modern Design Agency Homepage** built using **Next.js (App Router)** and **Tailwind CSS**.
The application was developed as part of the **Next.js Developer Internship evaluation task**.

The goal of this project is to demonstrate strong **frontend fundamentals, component-based architecture, responsive design, and clean coding practices**.

The homepage contains four major sections:

* Hero Section
* Services Section
* Portfolio Section
* Contact Section

The UI focuses on a **clean, professional, and responsive layout** suitable for a digital design agency.

---

## Tech Stack

* **Next.js 13+ (App Router)**
* **React Functional Components**
* **Tailwind CSS**
* **Next/Image for image optimization**
* **Vercel for deployment**

---

### Hero Section

## Installation and Setup

Follow these steps to run the project locally.

### 1. Clone the repository

```
git clone https://github.com/your-username/design-agency-homepage.git
```

### 2. Navigate to the project folder

```
cd design-agency-homepage
```

### 3. Install dependencies

```
npm install
```

### 4. Run the development server

```
npm run dev
```

The application will run at:

```
http://localhost:3000
```
